🚀 心理諮詢網站啟動指南
================================

📁 可用的啟動腳本：

1. 🎯 start-complete.bat (推薦)
   - 啟動完整網站（前端 + 後端 + 資料庫）
   - 適用：完整測試、演示、正常使用
   - 訪問地址：
     * 網站首頁：http://localhost:3000
     * 管理後台：http://localhost:8000/admin

2. ⚡ start-simple.bat
   - 僅啟動後端服務（後端 + 資料庫）
   - 適用：API 測試、後端開發
   - 訪問地址：
     * 後端 API：http://localhost:8000
     * 管理後台：http://localhost:8000/admin

3. 🚢 deploy.bat
   - 生產環境部署腳本
   - 適用：正式部署到伺服器

📋 首次使用步驟：
1. 確保 Docker Desktop 已啟動
2. 雙擊 start-complete.bat
3. 等待啟動完成
4. 創建管理員帳號：
   docker exec -it counseling_backend python manage.py createsuperuser
5. 瀏覽器打開 http://localhost:3000

🛠 常用指令：
- 查看運行狀態：docker-compose ps
- 查看日誌：docker-compose logs -f
- 停止服務：docker-compose down
- 重啟服務：docker-compose restart

⚠️ 注意事項：
- 需要 Docker Desktop 運行
- 首次啟動需要下載鏡像，可能較慢
- 8000、3000、3306 端口需要空閒
